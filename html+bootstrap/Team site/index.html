HoloRespository

The project we are working on is an implementation of the Holo Repository for 3D viewing on Hololens to help in surgical planning for operations. The data in surgery is required to be stored along the patient records. It should also have a feature for training aid.

Solution for our project

The project is to build a database, which has cloud data exchange technique providing the users with information about the patients along with 3D data for complex surgical operations.

So, we decide to use Maven to manage our repository , which actually is a Web App. And such app is developed by using React , and will be deployed through tomcat.

Feacture of our project

View 3D images of the different parts of the body

Able to view on HoloLens

Able to access patient records from different hospitals in one place

Encryption of the data

Map Holorepository entries with patient data.

    After gathering all the requirements from the our client , we have built up a basic prototype to show how our respository interact with users. Such prototype include the details that the users requested us along with in response to the heuristic evaluation. Since our project is actually a web app staff , so we have to design an relative simple but functional user interface for users.
